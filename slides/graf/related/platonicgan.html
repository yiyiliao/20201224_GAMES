## 3D Representations #### 3D Shape with Volumetric Rendering
PlatonicGAN [Henzler et al., ICCV 2019]
  • Multi-view consistent
  • Low image fidelity, high memory consumption
  • Another work is PlatonicGAN.
  • Instead of an abstract 3D feature they directly generate the 3D object from random noise.
  • This allows replace the learned projection with a volumetric rendering that does not contain any learnable parameters
  • As shown by the COLMAP reconstruction, this leads to multi-view consistent images
  • However, in this approach the 3D object is represented on coarse voxel grid
  • This results in visible artifacts. Further the memory footprint of the representation grows cubically with image size and thus prevents method to scale to high resolution
